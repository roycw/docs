With Code Mode you can view & edit your project’s raw code directly in Niles Code editor.

<Note>
  Editing code in Code Mode is a feature available for all plans, including the free plan!
</Note>

Now you can also Sync your Niles code with a local repo and keep working localy!, just use the Sync code button at the top bar:
